# Battle City

A reimagined 2D version of the NES game Battle City, built in Rust with [Bevy 0.18](https://bevyengine.org/).

## Requirements

- [Rust](https://www.rust-lang.org/tools/install) 1.95+

## Build & Run

```bash
# Debug build and run
cargo run

# Release build (optimized)
cargo build --release
./target/release/battle-city
```

## Controls

| Action | Key |
|--------|-----|
| Move | Arrow keys |
| Fire | Space |
| Ability | Left Shift |
| Pause | Escape |
